# 🪵 Nakama Landing Page

Landing page desarrollada para **Nakama**, una marca dedicada al diseño y fabricación de mobiliario a medida de alta calidad.

El objetivo del proyecto fue crear una experiencia visual elegante, minimalista y profesional, transmitiendo la identidad de la marca: precisión, diseño, oficio artesanal y tecnología aplicada a la fabricación.

---

## ✨ Características principales

- Diseño moderno inspirado en estudios de arquitectura e interiorismo.
- Hero dinámico con cambio automático de imágenes.
- Secciones informativas sobre filosofía de marca.
- Galería de proyectos con enfoque visual.
- Presentación de procesos de fabricación.
- Información sobre tecnología, materiales y terminaciones.
- Contacto directo mediante WhatsApp.
- Diseño responsive adaptable a diferentes dispositivos.

## 🚀 Instalación y ejecución

Clonar el repositorio:

```bash
git clone <repository-url>
```

Ingresar al proyecto:

```bash
cd nakama-landing
```

Instalar dependencias:

```bash
npm install
```

Ejecutar entorno de desarrollo:

```bash
npm run dev
```
